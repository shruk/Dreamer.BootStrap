Display Property

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can). div, h1, p, form, header, footer, secion

An inline element does not start on a new line and only takes up as much width as necessary. span, a, img

Position Property

The position property specifies the type of positioning method used for an element (static, relative, fixed, absolute or sticky).

static position

static position is not going to move even we have a left attributes, it is a default position

relative position

relative position can be changed based on top, buttom, left and right from its normal position
fixed position always stays in the same place even the page scrolled

absolute position is positioned relative to nearest positioned ancestor

Note: A "positioned" element is one whose position is anything except static.

This div element has position: absolute;

Sticky position is between fixed and relative

sticky can be relative until its given offset position is met in the viewport--become fixed
